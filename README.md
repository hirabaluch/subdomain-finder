# 🔍 Subdomain Finder

A fast, powerful, and CLI-based subdomain discovery tool built with Python, designed for reconnaissance and bug bounty hunting.

---

## 🚀 Features

* ⚡ Fast scanning using multithreading
* 🎯 CLI-based usage (professional)
* 📂 Custom wordlist support
* 💾 Saves results to file
* 🔍 Supports HTTP & HTTPS detection

---

## 🛠 Installation

```bash
git clone https://github.com/yourusername/subdomain-finder.git
cd subdomain-finder
pip install requests
```

## ⚡ Usage

```bash
python subdomain_finder.py -d example.com -w wordlist.txt
```
